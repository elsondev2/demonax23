  🚨 URGENT FIX - SMTP BLOCKED ON RENDER
═══════════════════════════════════════════════════════════════

PROBLEM: Render blocks SMTP port 587 → Emails timeout ❌

SOLUTION: Use Resend (5 minutes) ✅

═══════════════════════════════════════════════════════════════
  STEP 1: GET RESEND API KEY (2 minutes)
═══════════════════════════════════════════════════════════════

1. Go to: https://resend.com
2. Sign up (free account)
3. Click "API Keys" → "Create API Key"
4. Name: "de_monax-production"
5. Permission: "Sending access"
6. Click "Add"
7. COPY THE KEY (starts with "re_")
   ⚠️ Save it now! You can't see it again!

═══════════════════════════════════════════════════════════════
  STEP 2: ADD TO RENDER (2 minutes)
═══════════════════════════════════════════════════════════════

1. Go to: https://dashboard.render.com
2. Click your backend service: "demonax23"
3. Click "Environment" tab
4. Click "Add Environment Variable"

Add these 3 variables:

┌─────────────────────────────────────────────────────────────┐
│ RESEND_API_KEY                                              │
│ re_your_actual_api_key_here                                │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ EMAIL_FROM                                                  │
│ onboarding@resend.dev                                      │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ EMAIL_FROM_NAME                                             │
│ de_monax                                                   │
└─────────────────────────────────────────────────────────────┘

5. Click "Save Changes"
6. Wait 2-3 minutes for redeploy

═══════════════════════════════════════════════════════════════
  STEP 3: TEST IT WORKS (1 minute)
═══════════════════════════════════════════════════════════════

1. Go to your app
2. Sign up with email
3. Check inbox for verification code
4. Check Render logs for:
   ✅ "Email sent successfully via Resend"
