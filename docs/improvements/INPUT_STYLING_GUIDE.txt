TEXT INPUT STYLING GUIDE
========================

PROBLEM SOLVED
--------------
Text inputs looked like dark images in desktop view.
Now they have a clean, professional appearance.

NEW STANDARD INPUT CLASSES
---------------------------

For ALL text inputs:
  input input-bordered 
  bg-base-100 
  focus:bg-base-100 
  focus:outline-none 
  focus:ring-2 
  focus:ring-primary/20 
  focus:border-primary

For ALL textareas:
  textarea textarea-bordered 
  bg-base-100 
  focus:bg-base-100 
  focus:outline-none 
  focus:ring-2 
  focus:ring-primary/20 
  focus:border-primary 
  resize-none

For ALL select dropdowns:
  select select-bordered 
  bg-base-100 
  focus:bg-base-100 
  focus:outline-none 
  focus:ring-2 
  focus:ring-primary/20 
  focus:border-primary

For ALL labels:
  label-text font-semibold
